# GDSC Nexus

Empowering students to learn, build, and grow together. Join our community of passionate developers and innovators. Master cutting-edge technologies, collaborate on real projects, and launch your tech career with us.

## Quick Start

### Local Development

1.  **Clone the repository**
2.  **Install dependencies**: `npm install`
3.  **Start the development server**: `npm run dev`

## Technologies

- **Frontend**: Vite, React, TypeScript, Tailwind CSS, shadcn-ui
- **Backend**: NestJS, Prisma, PostgreSQL
- **Theming**: next-themes
- **Components**: Lucide-React, Radix UI

## Features

- **Project Management**: Showcase and collaborate on student projects.
- **Team Collaboration**: Dedicated spaces for different developmental tracks.
- **Event Tracking**: Stay updated with the latest workshops and hackathons.
- **General Chat**: Real-time communication with community members.
- **Dark Mode**: Personalized viewing experience for all members.
